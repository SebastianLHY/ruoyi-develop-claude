## 📚 项目资源概览

### 后端已有系统功能模块
| 模块 | 说明 | 位置 |
|------|------|------|
| ✅ 用户管理 | 管理后台的登录用户和对用户授权 | ruoyi-system/user/ |
| ✅ 角色管理 | 分配菜单权限，实现多角色权限控制 | ruoyi-system/role/ |
| ✅ 菜单管理 | 动态管理页面上的路由，配置系统菜单 | ruoyi-system/menu/ |
| ✅ 部门管理 | 管理组织架构及部门信息 | ruoyi-system/dept/ |
| ✅ 岗位管理 | 管理岗位信息及职责 | ruoyi-system/post/ |
| ✅ 字典管理 | 统一管理常用数据分类及系统字典 | ruoyi-system/dict/ |
| ✅ 参数设置 | 配置系统运行参数 | ruoyi-system/config/ |
| ✅ 通知公告 | 管理系统通知及公告信息 | ruoyi-system/notice/ |
| ✅ 日志管理 | 记录后台登录以及所有的操作日志 | ruoyi-system/monitor/log/ |
| ✅ 在线用户 | 监控和管理当前在线用户会话 | ruoyi-system/monitor/online/ |
| ✅ 代码生成 | 自动生成增删改查代码，支持多表操作 | ruoyi-generator/ |
